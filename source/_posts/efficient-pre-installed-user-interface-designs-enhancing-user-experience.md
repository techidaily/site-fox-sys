## Pre-install UI

The Pre-install UI is handled by the [EXE boostrapper](https://tools.techidaily.com/advancedinstaller/products/) and it requires the [Enhanced User Interface](https://tools.techidaily.com/advancedinstaller/products/) option.

![Important](https://cdn.advancedinstaller.com/svg/common/IconMessageInfo.svg)If the MSI inside runs with a UI level lower than this, the MSI UI level is used.

Dialogs are automatically added to this section after:

* adding pre-install prerequisites from the [Prerequisites Page - Installer Project](https://tools.techidaily.com/advancedinstaller/products/)
* selecting the "Display language selection dialog (for multi-language packages)" option from the [Languages Tab](https://tools.techidaily.com/advancedinstaller/products/)
* enabling [Multiple Instances Page](https://tools.techidaily.com/advancedinstaller/products/)

You can then customize these dialogs using the [Dialog Editor](https://tools.techidaily.com/advancedinstaller/products/).
